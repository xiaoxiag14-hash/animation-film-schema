Relevant Metadata Standards

External metadata standards used or referenced in this schema.

📚 Standards Overview

📘 Dublin Core Terms

Used for title, date, and description.

🌐 Schema.org

Used as a web-oriented reference for movie description.

🎥 EBUCore

Referenced for media and broadcasting-related concepts.

✨ anim

Used for animation-specific custom elements.
