# Animation Film Metadata Schema Website

This folder contains the final project website files.

## Files

- `index.html`: Main website page
- `style.css`: Website styling
- `animation.xsd`: XSD schema file
- `sample-your-name.xml`: Sample XML record 1
- `sample-spirited-away.xml`: Sample XML record 2
- `sample-spider-verse.xml`: Sample XML record 3

## How to publish

### Easy option: Netlify
1. Go to https://app.netlify.com/drop
2. Drag and drop this whole folder or the ZIP file.
3. Netlify will give you a public website URL.
4. Submit that URL to the class forum.

### GitHub Pages option
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Choose the main branch and root folder.
5. GitHub will generate a public website URL.

## Important after publishing

After the website is published, the direct XSD URL will usually be:

`https://YOUR-SITE-URL/animation.xsd`

You may replace `animation.xsd` in the XML sample records with the full public XSD URL if required by the instructor.
